关于 StudyHelper

StudyHelper 是一款面向广大学子打造的轻量级纯静态学习辅助工作台。我们在底层架构中严格遵循“去服务器、零网络接口、隐私彻底自控”的产品设计理念。

数据安全：100% 宿主内隔离
运行模式：完全静态单页沙箱
基础架构：原生 JS + Tailwind
开源分发：支持 GitHub Pages
